# Getting started

## Prerequisites

To use Inversify GraphQL, you need to have an Inversify HTTP server set up.
Please refer to the [HTTP Getting Started](../../../docs/introduction/getting-started) guide for instructions on how to set up an HTTP server.

## Install dependencies

Install the required packages:

<Tabs>
  <TabItem value="npm" label="npm">

```bash
npm install @inversifyjs/apollo-express @inversifyjs/apollo-core @apollo/server graphql
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
pnpm install @inversifyjs/apollo-express @inversifyjs/apollo-core @apollo/server graphql
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
yarn add @inversifyjs/apollo-express @inversifyjs/apollo-core @apollo/server graphql
```

  </TabItem>
</Tabs>
