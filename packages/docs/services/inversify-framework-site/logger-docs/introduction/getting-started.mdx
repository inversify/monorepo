# Getting started

`@inversifyjs/logger` is a powerful simple library to create loggers.


## Install dependencies

Install `@inversifyjs/logger` package:

<Tabs>
  <TabItem value="npm" label="npm">

```bash
  npm install @inversifyjs/logger
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
  yarn add @inversifyjs/logger
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
  pnpm install @inversifyjs/logger
```

  </TabItem>
</Tabs>
