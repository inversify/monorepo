# Getting started

## Install dependencies

To get started with Inversify HTTP, first install the required packages.

Begin by installing the `inversify` packages and `reflect-metadata`:

<Tabs>
  <TabItem value="npm" label="npm">

```bash
  npm install inversify reflect-metadata @inversifyjs/http-core
```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

```bash
  pnpm install inversify reflect-metadata @inversifyjs/http-core
```

  </TabItem>
  <TabItem value="yarn" label="yarn">

```bash
  yarn add inversify reflect-metadata @inversifyjs/http-core
```

  </TabItem>
</Tabs>

:::warning

Make sure to enable the [Experimental Decorators](https://www.typescriptlang.org/tsconfig/#experimentalDecorators) and [Emit Decorator Metadata](https://www.typescriptlang.org/tsconfig/#emitDecoratorMetadata) options in your `tsconfig.json`.

:::

## Choose an HTTP adapter

Inversify HTTP works with several HTTP frameworks. Choose the adapter that best fits your needs:

<Tabs>
  <TabItem value="express-v4" label="Express v4">

```bash
  npm install @inversifyjs/http-express-v4
```

  </TabItem>
  <TabItem value="express" label="Express">

```bash
  npm install @inversifyjs/http-express
```

  </TabItem>
  <TabItem value="fastify" label="Fastify">

```bash
  npm install @inversifyjs/http-fastify
```

  </TabItem>

  <TabItem value="hono" label="Hono">

```bash
  npm install @inversifyjs/http-hono
```

  </TabItem>

  <TabItem value="uwebsockets" label="uWebSockets.js">

```bash
  npm install @inversifyjs/http-uwebsockets
```

  </TabItem>
</Tabs>
